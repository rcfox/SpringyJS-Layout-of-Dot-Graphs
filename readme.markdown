## How To Use
Use the `gvpr` tool that ships with Graphviz.

Example: `gvgen -S3 | gvpr -f springy-js.gv`

## Note
This is extremely limited, and is not meant as a full replacement for dot. It's mostly just to show off how to use `gvpr`.
